## Описание эксперимента

В работе реализована среда навигации автономного дрона для варианта 2. Среда содержит анизотропный ветер, пять прямоугольных препятствий, ограниченный заряд батареи и зарядную станцию.

Состояние дрона включает относительное положение до зарядной станции, уровень заряда батареи и текущую скорость. Для применения табличных методов состояние дискретизируется.

В работе сравниваются три агента:

- **Rational Q-learning** — классический табличный Q-learning.
- **Prospect Q-learning** — агент с преобразованием вознаграждения по теории перспектив.
- **Risk-sensitive Q-learning** — риск-чувствительный агент на основе энтропийного критерия.

## Параметры эксперимента

- Выбранный вариант: Variant 2.
- Количество обучающих эпизодов для каждого агента: 15000.
- Количество тестовых эпизодов для каждого агента: 300.
- Стратегия выбора действий: ε-greedy.
- Тестовое значение ε: 0.005.
- Для воспроизводимости используется фиксированный seed = 42.

## Основные результаты

В результате работы программы строятся графики обучения, CDF времени достижения цели, тепловые карты финального заряда батареи и траектории движения дрона.

Также рассчитываются тестовые метрики:

- успешность;
- среднее число шагов;
- энергозатраты;
- столкновения;
- финальный заряд батареи.

Для статистического сравнения агентов используется Welch t-test.
