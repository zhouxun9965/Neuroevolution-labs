## Описание эксперимента

В работе реализован нейроэволюционный алгоритм WANN (Weight Agnostic Neural Network) для задачи управления в среде `CartPole-v1`.

Согласно варианту №26 используется алгоритм WANN без дополнительных ограничений на структуру сети. Основная идея WANN состоит в том, что эволюционирует не набор отдельных весов, а структура нейронной сети. При оценке все связи используют общий shared weight.

## Представление решения

Индивид WANN включает:

- входные узлы;
- скрытые узлы;
- выходные узлы;
- список соединений между узлами;
- функции активации скрытых узлов.

Для CartPole-v1 входная размерность равна 4, а число выходных действий равно 2.

## Основные параметры

- Среда: `CartPole-v1`.
- Основной seed: `42`.
- Дополнительный seed для проверки воспроизводимости: `123`.
- Размер популяции: `40`.
- Число поколений: `60`.
- Число fitness evaluations в основном эксперименте: `2400`.
- Shared weights: `[-2.0, -1.0, -0.5, 0.5, 1.0, 2.0]`.
- Максимальное число скрытых узлов: `12`.

## Проведённые эксперименты

В работе были выполнены следующие эксперименты:

1. Основное обучение WANN.
2. Проверка воспроизводимости на двух seed.
3. Сравнение с baseline Random Search.
4. Абляция: отключение мутации функций активации.
5. Гиперпараметрическое сравнение размера популяции.
6. Гиперпараметрическое сравнение вероятностей мутации.
7. Проверка устойчивости к шуму наблюдений.
8. Сохранение и загрузка лучшей найденной структуры.
9. Анализ неудачных режимов работы.

## Основные результаты

Основной WANN достиг максимального среднего reward `500.0` в среде CartPole-v1.

Два независимых запуска с seed `42` и `123` также достигли reward `500.0`, что подтверждает воспроизводимость результата.

Baseline Random Search получил только `259.33`, а версия без мутации активации получила `276.33`. Это показывает, что эволюционный отбор и мутация функций активации являются важными частями алгоритма.

При тесте устойчивости агент почти не потерял качество при шуме наблюдений `std = 0.05`, но при шуме `std = 0.10` средний reward снизился до `187.41`.
